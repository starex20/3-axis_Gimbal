# 3-axis_Gimbal
Atmega4809를 활용한 3축 짐벌 임베디드 시스템 프로젝트입니다.

<br/><br/>

## 사용 부품

| 상품명       | 수량 | 금액(원) | 비고     |
|-------------|------|--------|---------|
| ATmega4809  | 1    | 0   | 학교에서 제공    |
| MG966R 서보 모터  | 3    | 19,500      |   |
